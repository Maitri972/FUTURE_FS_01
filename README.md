# 🌟 Premium Developer Portfolio

A high-performance, visually stunning, and modern developer portfolio website tailored for Computer Engineering students. Built using advanced web technologies, fluid canvas particle layouts, and responsive interaction patterns.

---

## ✨ Features

- **Dynamic Loading Screen:** Seamless custom JavaScript preloader that handles resource tracking before rendering the layout.
- **Fluid Canvas Background:** Interactive particle background built with vanilla HTML5 Canvas API that scales dynamically based on theme rules.
- **Glassmorphic Glass UI:** Modern card structures utilizing high-end backdrop filters for a futuristic, frosted-glass effect.
- **Interactive 3D Card Hover:** Immersive 3D mouse-tracking inertial tilt physics applied to core competency and experience items.
- **Custom Cursor Pipeline:** Smooth-follow custom glowing ring cursor that dynamically reacts to interactive viewport elements.
- **Fluid Theme Control Engine:** Instant dark-to-light theme matrices with architectural rotation states on buttons.
- **Automated Text Engine:** Smart infinite character loop arrays for writing and deleting title profiles.

---

## 🛠️ Technology Stack

- **Structure & Layout:** HTML5 (Semantic Architecture)
- **Styling Framework:** CSS3 (Custom Variables, Advanced Layout Matrices, Glassmorphism)
- **Animation Suite:** GSAP (GreenSock Animation Platform) via CDN
- **Icons & Typography:** FontAwesome 6.4.0, Google Fonts (Space Grotesk & Plus Jakarta Sans)
- **Core Engine:** Vanilla JavaScript (ES6+ Modules, Canvas API, ScrollTrigger)
